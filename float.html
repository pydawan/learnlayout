홈 목차

float

레이아웃을 잡는 데 사용하는 또 하나의 CSS 프로퍼티는 바로 float입니다. float은 다음과 같이 이미지 주위를 텍스트로 감싸기 위해 만들어진 것입니다. 

img {
  float: right;
  margin: 0 0 1em 1em;
}
